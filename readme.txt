== Description ==

 If you want to migrate from VIVVO CMS (www.vivvo.net) to WordPress this plugin is right for you.

 If you feel that you are not skilled to do it yourself CodeHub offers commercial service for migrating (including images) your VIVVO CMS website to WordPress for $200 per website. Please contact us at development@codehub.us if you are interested.
= Supports =

 * It is tested on VIVVO versions 4.x and should work on most versions

= Features =

 * Import VIVVO Users
 * Import VIVVO Categories
 * Import VIVVO Tags
 * Import VIVVO Articles (as posts)

== Installation ==

1. Download, upload and install vivvo2wordpress.zip under Plugins >> Add New > Upload, and activate the wp-vivvo plugin.
2. From your Wordpress Dashboard, go to Tools > VIVVO 2 WordPress > HOWTO

That's it ... have fun!
